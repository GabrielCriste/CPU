The Fashion-MNIST Dataset.

The Fashion-MNIST Dataset.

Value parameters

root

Root directory of dataset where train-images-idx3-ubyte t10k-images-idx3-ubyte exist.

train

If true, creates dataset from train-images-idx3-ubyte, otherwise from t10k-images-idx3-ubyte.
